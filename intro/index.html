# 市场需求分析

当今社会交通迅速发展，许多城市停车时寻找车位困难，常会遇到这些情况：

  1. 到达停车场不知道何处有空位，在停车场内来回搜寻
  2. 发现停车场无剩余车位，不得不寻找下一个停车场

本软件为解决这些实际问题，提供目的地附近停车场位置及空余车位位置，为车主顺利找到停车位提供便利。

# 软件设计思路

TIP

数据均采用 Socket 协议进行传输。数据传输规范见 Socket 协议规范

体制内停车场车位分别安装占空探测传感器，通过停车场管理客户端将车位状态数据上传到服务器；提供给车主的手机 APP 可以实时获取就近的停车场及车位状态信息。

# 停车场管理客户端

  • 数据采集：通过单片机采集停车场车位占空探测传感器信号并发送给停车场管理客户端(使用模拟数据)
  • 数据上传：每隔一段时间上传一次停车场数据到数据分发服务器

# 数据分发服务器

  • 数据接收：接收并存储来自停车场管理客户端的停车场数据
  • 数据推送：根据来自手机 APP 的请求，筛选并推送停车场数据

# 手机 APP

  • 数据获取：用户选择就近有停车位的停车场，向数据分发服务器请求选中停车场的数据
  • 数据展示：处理来自数据分发服务器的数据并以合适的方式展示给用户
