# Socket 协议规范

# 定义

S 表示 数据分发服务器

C 表示 停车场管理客户端

M 表示 手机 APP

由 S 发送至 M 的数据包称之为 S2M 数据包，以此类推。

# 协议头(S2M)

WARNING

协议头 仅适用于 S2M 数据包

其他传输无需协议头

一个完整的 S2M 数据包由两条消息组成：

发送顺序 定义 释义
#1 协议头 长度 4 字节的十六进制数据，内容为包体的长度
#2 包体 数据包内容
// Message #1 Header
00 00 00 0F

// Mesage #2 Body
{"key": "demo"}
